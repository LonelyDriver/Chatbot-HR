## Chatbot that helps to apply to a company

# Dependencies
    
    Flask>= 1.1.2
    tensorflow >=2.3.1
    numpy >= 1.18.5
    keras >= 2.4.3

# Install

    pip install flask

    pip install tensorflow

Maybe you need to install libcudart10.1

    sudo apt-get install libcudart.10.1

    pip install keras
